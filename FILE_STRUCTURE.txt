================================================================================
                        RUNNING THE PROJECT
================================================================================

Terminal 1 (Backend):
$ cd backend
$ python -m venv venv
$ source venv/bin/activate

$ npm install
$ npm run dev

- Web UI: http://localhost:5173
- API Docs: http://localhost:8000/docs

CLI Usage:
$ python backend/cli.py states
$ python backend/cli.py causelist --state 1 --district 1 --complex 01 --today
